# Agents Context

## Repository shape
- `frontend/`: Vite + React dashboard
- `backend/`: FastAPI API/webhooks/loans/Agro-AI services
- `docs/`: product and demo documentation
- `supabase/`: database resources

## Root workflow commands
- `npm run setup:backend`
- `npm run setup:frontend`
- `npm run api`
- `npm run dev`
- `npm run test:backend`
- `npm run train:ai`

## Environment setup
- Use `backend/.env.example` as the backend environment template (`backend/.env`).
